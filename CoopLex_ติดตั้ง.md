## 📋 ขั้นตอนการติดตั้ง

### 1. เปิด Google Apps Script
- เปิด Google Sheet ของคุณ
- คลิกเมนู **Extensions** → **Apps Script**
- หน้าต่าง Apps Script จะเปิดขึ้น

### 2. ตั้งชื่อโปรเจกต์
- คลิกที่ชื่อ "Untitled project" (มุมบนซ้าย)
- เปลี่ยนชื่อเป็น **CoopLex**

### 3. สร้างไฟล์ Code.gs
- ลบโค้ดเดิมทั้งหมดใน `Code.gs`
- วางโค้ดจากไฟล์ **CoopLex_Code.gs** ทั้งหมด
- ⚠️ แก้ไข `ADMIN_PASSWORD` เป็นรหัสผ่านที่คุณต้องการ

### 4. สร้างไฟล์ Index.html
- คลิกปุ่ม **+** ข้างไฟล์ (Add a file)
- เลือก **HTML**
- ตั้งชื่อว่า `Index` (ไม่ต้องพิมพ์ .html)
- ลบเนื้อหาเดิม แล้ววางโค้ดจากไฟล์ **CoopLex_Index.html**

### 5. ตรวจสอบ Sheet Name
ใน `Code.gs` บรรทัดที่ 7:
```javascript
const SHEET_NAME = 'รวมข้อมูล'; // เปลี่ยนให้ตรงกับชื่อ Tab ใน Sheet ของคุณ
```

### 6. Deploy เป็น Web App
- คลิกปุ่ม **Deploy** (มุมบนขวา) → **New deployment**
- ประเภท: **Web app**
- Execute as: **Me**
- Who has access: **Anyone** (ให้ทุกคนดูได้)
- คลิก **Deploy**
- คัดลอก URL ที่ได้ → นั่นคือลิงก์เว็บไซต์ของคุณ!

### 7. ทดสอบ
- เปิด URL ที่ได้
- ทดสอบค้นหาข้อมูล
- คลิก "เข้าสู่ระบบ Admin" → ใส่รหัสผ่าน → ทดสอบเพิ่ม/แก้ไข/ลบ

## 💡 หมายเหตุ

- หากแก้ไขข้อมูลใน Sheet โดยตรง ให้ **Reload** หน้าเว็บ
- เมื่ออัปเดตโค้ดครั้งต่อไป ต้อง Deploy ใหม่ (Manage deployments → Edit version)
- แนะนำให้เปลี่ยน `ADMIN_PASSWORD` ก่อนใช้งานจริง
